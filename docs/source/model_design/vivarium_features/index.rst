================================================
Vivarium feature documentation
================================================

The purpose of this page is to document Vivarium feature requests made by the research team to be implemented by the software engineers. The infomation on this page may be related to *what* the reserachers would like the Vivarium framework to do and *how it will be communicated* between the research and engineering teams. Details related to implementation of such features may be provided in the Vivarium framework documentation.
